# gemini-chatbot-api
Gemini Chatbot API adalah proyek berbasis Node.js (Express.js) yang berfungsi sebagai middleware API untuk berinteraksi dengan Google Gemini (Generative AI). Proyek ini menyediakan endpoint RESTful yang dapat menerima input teks (prompt) dari pengguna dan mengembalikan respons yang dihasilkan oleh model Gemini AI secara real-time.
